1.start.bat and stop.bat should be depolyed under NULS 2.0 wallet programme root directory.
For example:
nuls-chainbox\NULS_WALLET\start.bat
nuls-chainbox\NULS_WALLET\stop.bat

2. .dll lib files are deployed under Libraries\CPP\Qt\5.12.3 of NULS 2.0 wallet programme root directory.
For example:
nuls-chainbox\NULS_WALLET\Libraries\CPP\Qt\5.12.3
